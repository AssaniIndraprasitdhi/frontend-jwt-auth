# Frontend JwtAuthDemo (Vue 3 + TypeScript)

ระบบ Login ด้วย JWT โดยใช้ Vue 3 + Vite + Pinia พร้อมจัดการ Token และ Route ป้องกัน

## Features

- Login / Register
- Protected Route
- JWT Auth (localStorage หรือ httpOnly Cookie)
- รองรับระบบ Role / Refresh Token

## Tech Stack

- Vue 3 + Vite
- TypeScript
- TailwindCSS
- Vue Router 4
- Pinia
- Axios

## Getting Started

```bash
npm install
npm run dev
